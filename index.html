Commande Vim:

  • :q ! : permet de sortir de vim
  • I : permet d'incruster du texte avant le curseur
  • A : permet d'ajouter du texte après le curseur
  • :wq : permet d'enregistrer un fichier et de sortir
  • x : permet d'effacer le caractère sous le curseur
  • dw : permet d'effacer un mot complet en ce placant au début de celui ci
  • d$ : permet d'effacer une ligne complète jusqu’à la fin
  • Nw : déplace de n mot vers l'avant
  • Ne : déplace de le curseur de n mots a la fin de celui ci
  • dNw : permet d'effacer n mots a partir du curseur
  • Ndd : permet d'effacer n ligne a partir du curseur, un simple dd efface une ligne
  • u : annule la dernier modif
  • maj u : annul la derniere ligne de modif
  • ctrl R : reviens en arriere
  • ce : permet d'afface le mot a la suite du curseur et de passer en mode insertion tout de suite
  • r suivi d'un lettre permet de remplacer la lettre sous le curseur
  • NG permet d'aller a la ligne N directement

Commandes Vim:

  • :q ! : permet de sortir de vim
  • I : permet d'incruster du texte avant le curseur
  • A : permet d'ajouter du texte après le curseur
  • :wq : permet d'enregistrer un fichier et de sortir
  • x : permet d'effacer le caractère sous le curseur
  • dw : permet d'effacer un mot complet en se plaçant au début de celui-ci
  • d$ : permet d'effacer une ligne complète jusqu’à la fin
  • Nw : déplace de n mot vers l'avant
  • Ne : déplace le curseur de n mots à la fin de celui-ci
  • dNw : permet d'effacer n mots à partir du curseur
  • Ndd : permet d'effacer n ligne à partir du curseur, un simple dd efface une ligne
  • u : annule la dernière modif
  • maj u : annule la dernière ligne de modif
  • ctrl R : revient en arrière
  • ce : permet d'effacer le mot à la suite du curseur et de passer en mode insertion tout de suite
  • r suivi d'une lettre : permet de remplacer la lettre sous le curseur
  • NG : permet d'aller à la ligne N directement
  • Ctrl-G : affiche votre position dans le fichier et l'état de celui-ci
  • Maj-G : vous place à la fin du fichier
  • :w : pour enregistrer les changements faits au fichier
